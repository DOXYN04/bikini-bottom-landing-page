# bikini-bottom-landing-page
A vibrant, highly interactive, and mobile-friendly web landing page featuring the world of SpongeBob SquarePants. Built with Tailwind CSS and vanilla Web Audio API for pure nostalgic fun!
# 🧽 Bikini Bottom - Interactive Landing Page

[![Aura - Just For Fun](https://img.shields.io/badge/Vibe-Just%20For%20Fun-yellow.svg)](#)
[![Tech - TailwindCSS](https://img.shields.io/badge/Tech-Tailwind%20CSS-blue.svg)](#)

Selamat datang di dunia bawah laut Bikini Bottom! Proyek ini adalah sebuah *interactive landing page* modern yang dirancang untuk membawa keseruan kartun SpongeBob SquarePants langsung ke dalam peramban web Anda. Proyek ini dibuat murni untuk hiburan (*just for fun*) sekaligus mengeksplorasi pembuatan aset interaktif tanpa *library* eksternal yang berat.

## 🚀 Fitur Utama

*   **🪼 Game Tangkap Ubur-Ubur (Jellyfishing):** Mini-game 30 detik berbasis *click/touch events* langsung di dalam halaman web.
*   **🍔 Simulator Krabby Patty:** Susun burger legendaris Anda sendiri secara interaktif dan dapatkan penilaian langsung dari Tuan Krabs.
*   **🔊 Audio Synth Tanpa File (Soundboard):** Menghasilkan efek suara gelembung pecah dan ketawa khas SpongeBob menggunakan **Web Audio API** bawaan browser (tanpa aset audio eksternal).
*   **🫧 Efek Gelembung Mengapung:** Latar belakang dinamis menggunakan CSS Animations yang memberikan nuansa seolah berada di dalam air.
*   **📱 Responsive & Mobile-Friendly:** Dioptimalkan sepenuhnya menggunakan Tailwind CSS untuk kenyamanan akses melalui *smartphone* maupun desktop.

## 🛠️ Teknologi yang Digunakan

*   **HTML5** (Struktur & Semantik)
*   **Tailwind CSS** (Gaya Visual & Tata Letak Responsif)
*   **Vanilla JavaScript** (Logika Game, Simulator Burger, & Manipulasi DOM)
*   **Web Audio API** (Sintesis Efek Suara *Real-time*)
